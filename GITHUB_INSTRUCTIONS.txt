🚀 КАК ЗАГРУЗИТЬ НА GITHUB (3 МИНУТЫ)

1️⃣ Установить Git (если не установлен):
   https://git-scm.com/download/win

2️⃣ Создать репозиторий на GitHub:
   https://github.com/new
   Имя: chess-helper
   Нажать "Create repository"

3️⃣ Выполнить команды в PowerShell:

cd c:\Users\user\Desktop\assist
git init
git config user.name "Ваше имя"
git config user.email "ваша@почта.com"
git add .
git commit -m "Initial commit: Chess helper v2.0"
git branch -M main
git remote add origin https://github.com/ВАША_ССЫЛКА/chess-helper.git
git push -u origin main

(Замените ВАША_ССЫЛКА на ваш GitHub username)

Готово! ✅
